# 🎨 Estudos de Frameworks CSS — Bootstrap & Tailwind

Um **framework CSS** é uma biblioteca de estilos prontos que evita escrever todo
o CSS do zero. Esta pasta estuda os **dois mais populares** — que resolvem o
mesmo problema com **filosofias opostas**.

## 📁 Arquivos

| Arquivo | O que é |
|---|---|
| [`bootstrap.html`](./bootstrap.html) | Bootstrap na prática: navbar, botões, card, grid de 12 colunas — filosofia de **componentes prontos**. |
| [`tailwind.html`](./tailwind.html) | Tailwind na prática: os mesmos componentes montados peça por peça — filosofia **utility-first**. |
| [`comparativo.md`](./comparativo.md) | Bootstrap **vs.** Tailwind: tabela, quando usar cada um e instalação. |

## ▶️ Como abrir

São arquivos HTML estáticos: **abra direto no navegador** (duplo clique) ou use
uma extensão como *Live Server* no VS Code. Ambos carregam o framework via **CDN**,
então precisam de **conexão com a internet** para funcionar.

## 🚀 Sugestão de estudo

1. Abra `bootstrap.html` e veja como poucas classes já entregam componentes.
2. Abra `tailwind.html` e veja os **mesmos** componentes montados com utilitários.
3. Leia o `comparativo.md` para consolidar as diferenças e saber **quando usar cada um**.
